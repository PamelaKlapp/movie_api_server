Movies App

Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameter Request Body data format Response Body data format
Get a list of movies /movies GET None None Array of JSON objects holding data about all movies
Get data about a single movie by title /movies/[movieTitle] GET :movieTitle None A JSON object holding data about a single movie, containing a Name, description, genre, director, image URL and whether it's feautured or not. Example:
{
title: 'Dune',
description: 'A noble family becomes embroiled in a war for control over the galaxy's most valuable asset while its heir becomes troubled by visions of a dark future.',
genre: 'Sci-Fi',
director:{
   name: 'Denis Villeneuve',
   birth: '03/08/1967'
   },
image: '../images/movie_images/dune.svg',
feautured: true
}
Get data about movies by genre /movies/genre/[genreType] GET :genreType None A JSON object holding data about movies, containing an especific genre. Example:
{
title: 'Dune',
description: 'A noble family becomes embroiled in a war for control over the galaxy's most valuable asset while its heir becomes troubled by visions of a dark future.',
genre: 'Sci-Fi',
director:{
   name: 'Denis Villeneuve',
   birth: '03/08/1967'
   },
image: '../images/movie_images/dune.svg',
feautured: true
}
Get data about a director /movies/director/[directorName] GET :directorName None A JSON object holding data about a director, containing an especific director name. Example:
director:{
name: 'Denis Villeneuve',
birth: '03/08/1967'
}
Add an user /users/register POST :register A JSON object holding data about a new user. Example:
{
name: "Martin Schmidt",
username: "MartinLovesMovies",
email: "martin@email.com"
} 		A JSON object holding data about a new user that was added, including an ID:
{
name: "Martin Schmidt",
username: "MartinLovesMovies",
email: "martin@email.com"
}
Update info from an user /users/update/[username] PUT :username None A text message indicating whether the user's username was succesfully update.
Allows users to add a movie to their List of Fvorites /users/id/favorites/[movieTitle] POST :movieTitle A JSON object holding data about the movie the user added. Example:
{
title: 'Kill Bill',
director: 'Quentin Tarantino'
} 		A text messages indicating the movie was added.
Remove a movie from a user's list /users/id/favorites/[movieTitle] DELETE :movieTitle None A text message indicating the movie was removed.
Remove an user from the app /users/[id] DELETE :id None A text message indicating the email of the user is not longer on the app.
